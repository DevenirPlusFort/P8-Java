# 类注释

```java
/***
 * 
 * @File  $file$
 * @Desc  
 * @Author codebetter
 * @Create $DATE$ $TIME$ 
 *
 * @ChangeList
 * ------------------------------------------------------------------------
 * Date					Editor				   ChangeReason			
 *                 
 */
```

# 类行注释

```java
/***
 * 
 * @File 
 * @Desc  
 * @Author codebetter
 * @Create ${YEAR}/${MONTH}/${DAY} ${TIME}
 *
 * @ChangeList
 * ------------------------------------------------------------------------
 * Date					Editor				   ChangeReason			
 *                 
 */
public class ${NAME} {
}
```

# 方法注释

